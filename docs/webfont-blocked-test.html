  • ネットワーク節約アプリ等で Web フォントのロードが制限されたケースをエミュレートします
  • DATA URI スキーム化した Web フォントを埋め込んだ CSS を読み込みます
  • この CSS のロードに失敗したり、Web フォントの判定に失敗した場合は、背景画像置換でフォールバックします
